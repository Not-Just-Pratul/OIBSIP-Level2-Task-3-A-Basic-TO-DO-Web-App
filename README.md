# Basic To-Do Web App

This repository contains a simple to-do web application built with HTML, CSS, and JavaScript.

## Project Structure

- `Task6/TODO/index.html` - Main HTML file for the app.
- `Task6/TODO/styles.css` - Styling for the to-do app.
- `Task6/TODO/app.js` - JavaScript logic for adding, completing, and removing tasks.

## How to Run

1. Open `Task6/TODO/index.html` in a web browser.
2. Add tasks using the input field and add button.
3. Manage tasks directly from the list.

## Notes

- This is a static front-end app and does not require a server.
- Works in modern browsers.
