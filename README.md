# Basic To-Do Web App

A clean and lightweight to-do web application built with HTML, CSS, and JavaScript. This project provides a simple interface for creating, managing, and organizing daily tasks efficiently.

## Features

- Add new tasks quickly
- Mark tasks as completed
- Remove tasks when no longer needed
- Simple and responsive user interface
- Fully client-side, requiring no backend setup

## Live Demo

View the deployed application here:
- https://a-basic-todo-web-app.netlify.app/

## Project Structure

- Task6/TODO/index.html - Main HTML structure of the application
- Task6/TODO/styles.css - Styling and visual design
- Task6/TODO/app.js - JavaScript logic for task interactions

## How to Run Locally

1. Open Task6/TODO/index.html in your browser.
2. Enter a task in the input field.
3. Click the add button to save it to the list.
4. Use the interface to complete or delete tasks as needed.

## Notes

- This is a static front-end application and does not require a server.
- It works well in modern web browsers.
